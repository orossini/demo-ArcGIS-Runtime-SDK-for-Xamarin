Demo d'une appli Xamarin.Forms avec ArcGIS Runtimes SDK for Xamarin.
Ici seules les versions Android et UWP de l'appli sont disponibles (il est également possible de créer une appli iOS).
Cette application basique permet de monter l'utilisation d'ArcGIS Runtime SDK for Xamarin avec Xamarin.Forms
L'application se connecte à ArcGIS Online pour récupérer la liste des basemaps existantes pour permettre à l'utilisateur de changer la basemap courante.
La liste s'affiche dans un composant d'UI différent en fonction de l aplateforme cible (Android ou UWP).
